# V0.25.0 — Docker 部署 实现方案

> **Goal:** Dockerfile + docker-compose.yml，将管线容器化，支持 GPU 直通、模型挂载、一键部署。
>
> **Architecture:**
> - `Dockerfile`: Python 3.11 + pipeline 脚本
> - `docker-compose.yml`: pipeline + ComfyUI + (可选) Ollama

---

## Docker 架构

```
docker-compose.yml
├── comfyui              (ghcr.io/comfyanonymous/comfyui:latest)
│   ├── ports: 8188
│   ├── volumes: models/, input/, output/
│   └── deploy: GPU 直通
│
├── pipeline             (build: .)
│   ├── depends_on: comfyui
│   ├── volumes: outputs/
│   └── env: COMFY_URL=http://comfyui:8188/prompt
│
└── ollama               (ollama/ollama:latest)
    ├── ports: 11434
    └── volumes: ollama_models/
```

## Task 1: Dockerfile

```dockerfile
FROM python:3.11-slim

WORKDIR /app

# 安装依赖
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt

# 拷贝管线脚本
COPY agents/ agents/
COPY workflows/ workflows/
COPY scripts/ scripts/

# 环境变量
ENV COMFY_URL=http://comfyui:8188/prompt
ENV PYTHONPATH=/app/agents

# 入口
ENTRYPOINT ["python", "-m", "agents"]
CMD ["--help"]
```

## Task 2: docker-compose.yml

```yaml
version: "3.8"

services:
  comfyui:
    image: ghcr.io/comfyanonymous/comfyui:latest
    ports:
      - "8188:8188"
    volumes:
      - comfyui_models:/workspace/ComfyUI/models
      - comfyui_input:/workspace/ComfyUI/input
      - comfyui_output:/workspace/ComfyUI/output
    environment:
      - NVIDIA_VISIBLE_DEVICES=all
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: all
              capabilities: [gpu]
    restart: unless-stopped

  pipeline:
    build: .
    depends_on:
      - comfyui
    volumes:
      - pipeline_outputs:/app/outputs
    environment:
      - COMFY_URL=http://comfyui:8188/prompt
    stdin_open: true
    tty: true

  ollama:
    image: ollama/ollama:latest
    ports:
      - "11434:11434"
    volumes:
      - ollama_models:/root/.ollama
    deploy:
      resources:
        reservations:
          devices:
            - driver: nvidia
              count: all
              capabilities: [gpu]
    restart: unless-stopped

volumes:
  comfyui_models:
  comfyui_input:
  comfyui_output:
  pipeline_outputs:
  ollama_models:
```
