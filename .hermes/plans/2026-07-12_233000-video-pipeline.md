# V0.24.0 — 视频生成管线（Wan2.1）实现方案

> **Goal:** `python -m agents video "prompt"` — Wan2.1 Text-to-Video / Image-to-Video，补齐最大市场需求缺口。
>
> **Architecture:** 新建 `agents/go_video.py`，程序化构建 Wan2.1 视频工作流。
>
> **已有模型:** `wan2.2_ti2v_5B_fp16.safetensors` (9.5GB) + `wan2.2_vae.safetensors` (1.3GB) + `umt5_xxl_fp8` CLIP

---

## 用户交互

```
# Text-to-Video
python -m agents video "赛博朋克城市，霓虹灯闪烁，雨夜" --frames 49
python -m agents video "prompt" --frames 81 --fps 15 --width 848 --height 480

# Image-to-Video
python -m agents video "人物行走" --ref start.png --frames 49

# 预览
python -m agents video "prompt" --dry-run
```

## Wan2.1 工作流节点结构

```
UNETLoader(1) ──→ KSampler(5)[0]
                      │
CLIPLoader(2) ──→ CLIPTextEncode(3) ──→ KSampler(5)[1] (positive)
                                  └──→ KSampler(5)[2] (negative, zero)

VAELoader(4) ──→ VAEDecode(6)[0]

EmptyLatentVideo(7) ──→ KSampler(5)[3]

KSampler(5) ──→ VAEDecode(6)

VAEDecode(6) ──→ VideoCombine(8) ──→ .mp4/.webm
```

| 节点 | 类型 | 说明 |
|------|------|------|
| 1 | `UNETLoader` | 加载 wan2.2_ti2v_5B_fp16.safetensors |
| 2 | `CLIPLoader` | 加载 umt5_xxl_fp8 (type=wan) |
| 3 | `CLIPTextEncode` | 正向/负向提示词编码 |
| 4 | `VAELoader` | 加载 wan2.2_vae.safetensors |
| 5 | `KSampler` | 视频采样（步数/CFG 影响大） |
| 6 | `VAEDecode` | 逐帧解码 |
| 7 | `EmptyLatentVideo` | 视频潜空间（宽/高/帧数/批次） |
| 8 | `VideoCombine` | 合成视频输出 |

## Task 1: 创建 go_video.py

**核心函数:**

```python
WAN_MODELS = {
    "unet": "wan2.2_ti2v_5B_fp16.safetensors",
    "clip": "umt5_xxl_fp8_e4m3fn_scaled.safetensors",
    "vae": "wan2.2_vae.safetensors",
}


def build_video_workflow(
    prompt: str,
    *,
    negative: str = "",
    seed: int = -1,
    steps: int = 30,
    cfg: float = 7.0,
    width: int = 848,
    height: int = 480,
    frames: int = 49,
    fps: int = 15,
    ref_image: str | None = None,
    prefix: str = "wan_video",
) -> tuple[dict, int]:
    """构建 Wan2.1 Text-to-Video / I2V 工作流。"""
    seed_actual = seed if seed != -1 else random.randint(1, 2**48 - 1)
    wf = {}
    
    wf["1"] = {"class_type": "UNETLoader", "inputs": {"unet_name": WAN_MODELS["unet"],
        "weight_dtype": "default"}}
    wf["2"] = {"class_type": "CLIPLoader", "inputs": {"clip_name": WAN_MODELS["clip"],
        "type": "wan"}}
    wf["3a"] = {"class_type": "CLIPTextEncode", "inputs": {"text": prompt, "clip": ["2", 0]}}
    wf["3b"] = {"class_type": "CLIPTextEncode", "inputs": {"text": negative or "", "clip": ["2", 0]}}
    wf["4"] = {"class_type": "VAELoader", "inputs": {"vae_name": WAN_MODELS["vae"]}}
    wf["5"] = {"class_type": "KSampler", "inputs": {"seed": seed_actual, "steps": steps, "cfg": cfg,
        "sampler_name": "euler", "scheduler": "normal", "denoise": 1.0,
        "model": ["1", 0], "positive": ["3a", 0], "negative": ["3b", 0],
        "latent_image": ["7", 0]}}
    wf["6"] = {"class_type": "VAEDecode", "inputs": {"samples": ["5", 0], "vae": ["4", 0]}}
    wf["7"] = {"class_type": "EmptyLatentVideo", "inputs": {"width": width, "height": height,
        "frames": frames, "batch_size": 1}}
    wf["8"] = {"class_type": "VideoCombine", "inputs": {"images": ["6", 0],
        "frame_rate": fps, "filename_prefix": prefix, "format": "video/h264-mp4"}}
    
    return wf, seed_actual
```
