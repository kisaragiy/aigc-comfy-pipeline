## 问题 1: Prompt 兜底

当前 `optimize_prompt()` 降级：Ollama 不可用 → 返回 raw input（如"画个好看女孩子"）
修复：Ollama 不可用 → 模板拼接英文 tag

### 模板兜底逻辑

```python
def _fallback_prompt(user_input: str) -> str:
    """Ollama 不可用时的模板兜底。"""
    # 检测关键词
    user_lower = user_input.lower()
    
    # 构建基础英文标签
    style = "anime style, anime coloring"
    quality = "masterpiece, best quality, ultra detailed, 8k"
    camera = "cowboy shot, upper body, looking at viewer"
    
    # 关键词匹配
    if any(w in user_lower for w in ["风景", "landscape", "山水", "城市", "建筑"]):
        camera = "wide shot, landscape, scenery, cityscape"
    if any(w in user_lower for w in ["全身", "full body", "站立", "站姿"]):
        camera = "full body, standing, dynamic pose"
    if any(w in user_lower for w in ["特写", "close up", "脸部", "face"]):
        camera = "close-up, face focus, detailed face"
    if any(w in user_lower for w in ["夜景", "night", "霓虹", "neon"]):
        style += ", night scene, neon lighting, dark atmosphere"
    
    # 直接使用用户输入作为主体描述（保留英文部分）
    english_tags = " ".join(w for w in user_input.split() if w.isascii())
    subject = english_tags if english_tags else user_input
    
    return f"{quality}, {style}, {subject}, {camera}, detailed background, soft lighting"
```

## 问题 2: 质量验证

`python -m agents validate --image out.png --prompt "prompt text"`

### 评分维度

| 维度 | 方法 | 权重 |
|------|------|------|
| CLIP Score | transformers.CLIPModel 计算图文相关性 | 40% |
| 人脸检测 | YOLOv8 face_yolov8m.pt — 检测崩脸 | 30% |
| 手部检测 | YOLOv8 hand_yolov8s.pt — 检测崩手 | 20% |
| 图像质量 | 亮度/对比度/锐度基础检查 | 10% |

### 输出

```
$ python -m agents validate --image out.png --prompt "a cat"

📊 质量评估: out.png
  CLIP Score:  0.32  (图文相关性)
  人脸检测:    ✅ 正常 (1 张脸)
  手部检测:    ⚠️ 异常 (7 根手指)
  图像质量:    ✅ 正常 (亮度/对比度 OK)
  ─────────────────────
  综合评分:    B (可接受)
```

```python
# 在 SIX_DIMENSION_TEMPLATE 之后、optimize_prompt 之前加：

def _fallback_prompt(user_input: str) -> str:
    """Ollama 不可用时的模板兜底。关键词匹配 → 英文 tag 模板。"""
    from .. import re  # noqa
    
    quality = "masterpiece, best quality, ultra detailed, 8k"
    base_style = "anime style, anime coloring"
    camera = "cowboy shot, upper body, looking at viewer"
    lighting = "soft lighting, natural light"
    bg = "detailed background"
    
    keywords = {
        "风景": {"camera": "wide shot, landscape, scenery, cityscape",
                 "bg": "scenic background, mountains, city"},
        "城市": {"camera": "wide shot, cityscape",
                 "lighting": "neon lighting, street lights, urban atmosphere"},
        "夜景": {"camera": "night scene",
                 "lighting": "moonlight, street lamps, dark atmosphere"},
        "全身": {"camera": "full body, standing, dynamic pose"},
        "半身": {"camera": "upper body, cowboy shot, waist up"},
        "特写": {"camera": "close-up, face focus, extreme close-up"},
        "战斗": {"style": "action scene, dynamic", "camera": "dynamic angle, action pose"},
        "海边": {"bg": "beach, ocean, seaside, waves, sand"},
        "教室": {"bg": "classroom, desk, blackboard, school interior"},
    }
    
    for kw, overrides in keywords.items():
        if kw in user_input:
            for key, val in overrides.items():
                if key == "camera": camera = val
                elif key == "lighting": lighting = val
                elif key == "bg": bg = val
                elif key == "style": base_style = val
    
    return f"{quality}, {base_style}, {user_input}, {camera}, {bg}, {lighting}"
```
