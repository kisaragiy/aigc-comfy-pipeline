# V0.18.0 — 工程化测试 + CI 实现方案

> **Goal:** 核心模块单元测试 + GitHub Actions CI，让管线工程化水平上一个台阶。
>
> **Architecture:** `tests/` 目录 + pytest + GitHub Actions
>
> **Tech Stack:** pytest, pytest-mock, GitHub Actions (ubuntu-latest, Python 3.11)

## Task 6: test_sweep.py

```python
"""测试 go_sweep 的核心函数。"""

from go_sweep import expand_grid, build_sweep_label


class TestExpandGrid:
    def test_single_key(self):
        grid = {"steps": [20, 30]}
        result = expand_grid(grid)
        assert len(result) == 2
        assert result == [{"steps": 20}, {"steps": 30}]

    def test_multi_key(self):
        grid = {"steps": [20, 30], "cfg": [1.0, 2.0]}
        result = expand_grid(grid)
        assert len(result) == 4
        assert {"steps": 20, "cfg": 1.0} in result
        assert {"steps": 30, "cfg": 2.0} in result

    def test_empty(self):
        assert expand_grid({}) == [{}]

    def test_single_value_lists(self):
        grid = {"steps": [20], "cfg": [1.0]}
        assert len(expand_grid(grid)) == 1


class TestBuildSweepLabel:
    def test_single_param(self):
        assert build_sweep_label({"steps": 20}) == "steps20"

    def test_multi_param(self):
        label = build_sweep_label({"steps": 20, "cfg": 1.0})
        assert "steps20" in label
        assert "cfg1.0" in label
```

## Task 7: CI 配置

`.github/workflows/test.yml`:

```yaml
name: Test

on: [push, pull_request]

jobs:
  test:
    runs-on: ubuntu-latest
    strategy:
      matrix:
        python-version: ["3.11"]

    steps:
    - uses: actions/checkout@v4
    
    - name: Set up Python ${{ matrix.python-version }}
      uses: actions/setup-python@v5
      with:
        python-version: ${{ matrix.python-version }}
    
    - name: Install dependencies
      run: |
        python -m pip install --upgrade pip
        pip install pytest
        pip install -r requirements.txt
    
    - name: Install project in dev mode
      run: pip install -e .
    
    - name: Run tests
      run: |
        cd agents && python -m pytest ../tests -v
```
